# WebSkript

Ein didaktischer Editor zur Einführung in algorithmisches Denken und Programmierung.

## Autor
Justus A. Weber

## Zweck
Das Programm dient dem Einsatz im Informatikunterricht und unterstützt:
- strukturiertes Programmieren
- Visualisierung durch Struktogramme
- einfache und motivierende Zugänge
